# Socket 통신

## Socket 통신 vs HTTP 통신

| 유형        | 설명                                                               |
| ----------- | ------------------------------------------------------------------ |
| Socket 통신 | 클라이언트와 서버가 서로에게 요청을 보낼 수 있는 방식, 양방향 통신 |
| HTTP 통신   | 클라이언트의 요청이 있을 때 서버가 응답하는 방식, 단방향 통신      |
