## HTTP API vs REST API

> HTTP API와 REST API는 거의 같은 의미로 사용하고 있다.
> 정확한 차이점으로 REST API는 HTTP 프로토콜을 따르면서 아래의 4가지 가이드 원칙을 지켜야 한다.
>
> 1. 자원의 식별
> 2. 메세지를 통한 리소스 조작
> 3. 자기서술적 메세지
> 4. 애플리케이션의 상태에 대한 엔진으로서 하이퍼미디어(HATEOAS)
>
> 우아한형제들 김영한 개발이사 인용
